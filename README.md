# Excel_Dashboard
Sales dashboard with creating the KPIs and trends in monthly sales.
Here we have sample sales data and we are going to create:
1. Total Sales, Total Orders, Total quatity and Total returns creating KPIs.
2. Pie chart for the percent of compleation of the product.
3. Filter and Slicer for the category, channel and Date.
4. Using the line chart for the monthly sales trend.
